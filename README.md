## ‚öôÔ∏è Installation & Set Up

1. Fork and clone this repo

    ```sh
    git clone https://github.com/mustaquenadim/mustaquenadim.github.io.git
    ```

2. Install and use the correct version of Node using [NVM](https://github.com/nvm-sh/nvm)

   ```sh
   nvm install
   ```

3. Install dependencies

   ```sh
   npm install
   ```

4. Start the development server

   ```sh
   npm run dev
   ```

## üõ†Ô∏è Building and Running for Production

1. Generate a full static production build

   ```sh
   npm run build
   ```

2. Preview the site as it will appear once deployed

   ```sh
   npm run start
   ```

## üöÄ Deploy

This site is configured for static export and can be deployed to any static hosting provider like GitHub Pages, Vercel, or Netlify.

For Vercel:
```sh
npx vercel
```

For static export (GitHub Pages):
```sh
npm run build
# Output will be in the 'out' directory
```
